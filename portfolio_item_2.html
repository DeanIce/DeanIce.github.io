A C# Monogame Project: Mario and Kirby

Monogame, DevOps, 2D Game design

In spring of 2021 I was a part of a team designated to recreate the first level of Super Mario bros using MonoGame and C#. I created sprite animations as well as handeled all collision detection and HUD updates. We stuck to strict pre-designed PBI's, and were then tasked to expand upon the game in a meaningful way. My team and I however had great chemistry and decided to create our own Scrum and create the first level of Kirby's Adventure.

We desinged a lengthy PBI document detailing the features we planned on adding, and tracked our progress in Azure DevOps. I led team meetings and created enemy animations as well as designed the level switch system, defeat and victory screens. I also developed an important skill to learn when to call a project finished. This was the project that solidifed my love for game design and you can find the first level using the link below!
